  - Selection sort

  - Merge sort
  - Quick sort

## possible next implementations:

# How to execute this code
You will need the following tools:
- CMake

## Step 1
Create a folder in the root of the project called.

## Step 2
In the folder you have just created, run the following commands:
```{bash}
cmake ..
cmake --build .
```
